Система управления студентами и курсами на FastAPI с SQLite базой данных.


- Управление студентами (добавление, просмотр, удаление)
- Управление курсами (создание, просмотр, удаление)  
- Запись студентов на курсы
- REST API для интеграции
- Веб-интерфейс

pip install -r requirements.txt
